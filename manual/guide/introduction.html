        # エディター

        Structuring-Editor (opens new window)

        # 前提条件

        対応ブラウザ：Chrome

        # 用語定義

        1. レコードアイテム
          Structuring-Editorではレコード群あるいは1つのレコードを レコードアイテム と定義しています

          レコードアイテムは タイトル部 と データ部 で構成されます。

        2. タイトル部
          タイトル部はレコードアイテムのタイトルを表現します。

          タイトル部ではとをクリックするとレコードアイテムそのもの対する操作の項目が表示されます。

        3. データ部
          データ部はタイトル部の情報に紐づくデータを表現します。

          データ部では右側にあるを押すことでレコードアイテムの削除を行うことができます。
          また、をクリックするとデータ部セルのテキストモードを選択する項目が表示されます。
